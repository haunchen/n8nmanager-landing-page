隱私權政策

生效日期：2025 年 6 月 18 日

1. 引言

本隱私權政策旨在說明「n8nManager」（以下簡稱「本應用程式」）如何收集、使用、儲存和保護您的資訊。本應用程式是一款專為管理 n8n 伺服器而設計的工具，不旨在收集或分享您的個人身份資訊。

4. 資訊的儲存與安全

您的 n8n 伺服器 URL 和 API 金鑰會安全地儲存在您的裝置上。URL 使用 SharedPreferences 儲存，而敏感的 API 金鑰則使用 flutter_secure_storage 進行加密儲存，以提供額外的保護。

所有與 n8n 伺服器的通訊都強制透過 HTTPS 進行加密，以保護數據在傳輸過程中的安全。如果 URL 未使用 HTTPS，應用程式將提示錯誤。

無雲端儲存：本應用程式不會將您的 n8n 伺服器連線資訊或任何從 n8n 伺服器獲取的數據上傳到任何雲端服務或第三方伺服器。所有數據處理和儲存均發生在您的本地裝置上。

5. 資訊的分享

本應用程式不會將您的任何資訊（包括 n8n 伺服器連線資訊或從 n8n 伺服器獲取的數據）分享、出售或出租給任何第三方。

6. 第三方服務連結

本應用程式可能包含指向外部網站的連結，例如用於顯示隱私權政策或關於頁面中開發者的部落格連結。這些外部網站有其自己的隱私權政策，我們鼓勵您查閱。我們對這些網站的內容、隱私權政策或實踐不承擔任何責任。

7. 您的選擇與控制

您可以隨時透過應用程式的設定頁面修改或刪除您儲存的 n8n 伺服器連線資訊。清除這些設定將從您的裝置中移除相關數據。

8. 本政策的變更

我們可能會不時更新本隱私權政策。任何變更將在本頁面發布，並附帶新的生效日期。建議您定期查閱本政策以了解任何更新。
